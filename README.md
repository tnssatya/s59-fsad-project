# s59-fsad

Monorepo root for the frontend app in `frontend/`.

## Requirements

- Node.js `20.19.0` or later
- npm `10` or later

## Quick start

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm run build
```

This runs the frontend production build and outputs files in `frontend/dist`.

## Lint

```bash
npm run lint
```
